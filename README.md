# MyIDPet

## Sobre o Projeto
O MyIDPet é uma plataforma digital desenvolvida como atividade acadêmica do curso de Análise e Desenvolvimento de Sistemas – UEMA.

O projeto tem como objetivo conectar tutores de animais domésticos e rurais a serviços especializados, promovendo interação social, segurança e divulgação de adoção e animais desaparecidos.

## Objetivos do Sistema
- Cadastro de usuários
- Cadastro de pets
- Publicação de postagens
- Divulgação de serviços pet
- Área para adoção de animais

## Ferramentas Utilizadas
- Trello – Planejamento do projeto
- Figma – Protótipo da interface
- dbdiagram.io – Modelagem do banco de dados
- GitHub – Documentação e versionamento
